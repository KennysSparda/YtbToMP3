# Conversor de Vídeo para MP3

Este é um script em Python que converte arquivos de vídeo para o formato MP3. Ele utiliza a biblioteca MoviePy para realizar a conversão.

## Executáveis

No diretório "dist" deste projeto, você encontrará dois executáveis:

1. `YoutubeDownload.exe`: Este executável permite que você selecione um arquivo de vídeo e converta-o para MP3 usando uma interface gráfica simples.

2. `ConvertToMP3.exe`: Este executável realiza a conversão de vídeo para MP3 a partir de um arquivo pré-selecionado sem a necessidade de interação com uma interface gráfica.

## Pré-requisitos

Antes de executar os executáveis, você precisa ter o Python instalado no seu sistema. 

Siga as etapas abaixo para baixar e instalar o Python:

1. Acesse o site oficial do Python em https://www.python.org.
2. Clique no botão "Downloads" no menu principal.
3. Na página de downloads, role para baixo até encontrar a seção "Python Releases for Windows".
4. Clique no link para baixar a versão mais recente do Python para Windows.
5. Execute o instalador baixado e siga as instruções para concluir a instalação. Certifique-se de marcar a opção "Add Python to PATH" durante o processo de instalação.

## Como usar

Para converter um arquivo de vídeo para MP3, siga estas etapas:

1. Execute o executável correspondente ao seu caso: `YoutubeDownload.exe` ou `ConvertToMP3.exe`.
2. Siga as instruções exibidas na interface gráfica (no caso do `YoutubeDownload.exe`) ou aguarde a conclusão da conversão (no caso do `ConvertToMP3.exe`).
3. Após a conversão, o arquivo de saída em formato MP3 será salvo no mesmo diretório do arquivo de vídeo original.

Certifique-se de ter os arquivos de vídeo em um formato suportado (por exemplo, .mp4, .3gpp).

Se você encontrar algum problema durante a execução dos executáveis ou precisar de assistência adicional, sinta-se à vontade para entrar em contato.
